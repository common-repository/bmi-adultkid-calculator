We have developed a free Wordpress plugin with our BMI Calculator that you can use completely free of charge.

== Description ==

The most complete free BMI calculator for WordPress with built-in translations and as the only plugin we include BMI calculation for adults and children based on the WHO BMI classification.

== Installation ==


1. Upload `bmi-calculator.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Generate a shortcode on the plugin page and paste it on your page
